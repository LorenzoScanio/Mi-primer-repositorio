## Descripción

Proyecto minimalista que contiene una página `index.html` y estilos en `style.css`. Ideal para practicar HTML/CSS básico y como punto de partida para añadir interactividad o backend.

## Características

- Plantilla HTML básica
- Hoja de estilos CSS separada
- Estructura de proyecto simple y clara

## Demostración

Abre `index.html` en tu navegador para ver la página de ejemplo.

## Instalación

1. Clona el repositorio:

```
git clone <URL-del-repositorio>
```

2. Abre `index.html` en un navegador web. No requiere servidor para la versión estática.

## Uso

- Edita `index.html` para cambiar el contenido.
- Modifica `style.css` para ajustar estilos.

Si deseas probar con un servidor local (opcional):

```
# con Python 3
python -m http.server 8000

# luego abrir http://localhost:8000
```
